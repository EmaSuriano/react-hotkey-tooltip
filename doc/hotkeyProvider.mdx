# HotkeyProvider

The idea of this component is to tell to all the Hotkeys when to display the tooltip and also how it has to look like. Therefore, just by having one `HotkeyProvider` in your application is enough!

<Props of={HotkeyProvider} />

<Playground>
  <HotkeyProvider>
    <Hotkey onPress="focus" combination="a">
      <button>Focus me!</button>
    </Hotkey>
    <Hotkey onPress="focus" combination="b">
      <button>Focus me!</button>
    </Hotkey>
    <Hotkey onPress="focus" combination="c">
      <button>Focus me!</button>
    </Hotkey>
  </HotkeyProvider>
</Playground>

## Advanced Usage

### Changing tooltip combination

<Playground>
  <HotkeyProvider tooltipCombination="shift+ctrl+h">
    <Hotkey onPress="focus" combination="a">
      <button>Focus me!</button>
    </Hotkey>
    <Hotkey onPress="focus" combination="b">
      <button>Focus me!</button>
    </Hotkey>
    <Hotkey onPress="focus" combination="c">
      <button>Focus me!</button>
    </Hotkey>
  </HotkeyProvider>
</Playground>

### Disabling Hotkeys

<Playground>
  <HotkeyProvider disabled>
    <Hotkey onPress="focus" combination="q">
      <button>Can't Focus me :(</button>
    </Hotkey>
    <Hotkey onPress="focus" combination="w">
      <button>Me neither ...</button>
    </Hotkey>
  </HotkeyProvider>
</Playground>

### Theming tooltip

<Playground>
  <HotkeyProvider tooltipCombination="shift+s">
    <Hotkey
      onPress="focus"
      combination="s"
      tooltipOptions={{
        theme: 'light',
        size: 'big',
        arrow: true,
      }}
    >
      <button>Special tooltip!</button>
    </Hotkey>
  </HotkeyProvider>
</Playground>
